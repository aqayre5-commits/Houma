# Houma

## Purpose
Day-one-ready local administrative guidance website for Casablanca, Rabat, and Tanger.

## Stack
- Next.js App Router
- Tailwind CSS
- Supabase
- Vercel Analytics
- Vercel Speed Insights

## Run
```bash
npm install
npm run dev
```

## Required env
```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.tld
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
NEXT_PUBLIC_ENABLE_ADS=false
NEXT_PUBLIC_ENABLE_SPONSORS=true
```

## Included docs
See `/docs` for concept gate, route map, SEO pack, analytics pack, QA pack, and release checklist.

## v1 limits
- 3 cities only
- 5 services only
- no auth
- no billing
- no booking
