# component_definitions.md

## Component Definitions
| Component Name | Purpose | Props | Variants | States | Accessibility | Responsive Behavior | Used On Pages | Monetization Relevance | Analytics Hooks | Acceptance Criteria |
|---|---|---|---|---|---|---|---|---|---|---|
| `SearchHero` | Query-led homepage entry | `title:string`, `subtitle:string`, `cities:City[]`, `services:Service[]` | homepage / compact | loading, success, error | labelled input, keyboard submit, visible focus | full width mobile | homepage | indirect | `click_primary_cta`, `search_filter_apply` | Search submits city/service combinations |
| `CityCard` | Open city page | `city:CitySummary` | default / featured | loading, success | link card with descriptive label | 1 per row on mobile | homepage, city index | indirect | `click_secondary_cta` | Card links to city route |
| `ServiceCard` | Open service page or city-service page | `service:ServiceSummary`, `citySlug?:string` | default / featured | loading, success | heading hierarchy intact | stacked mobile | homepage, city page, service index | indirect | `click_secondary_cta` | Card chooses correct target route |
| `QuickPathSwitcher` | Toggle city-first vs service-first flow | `mode:'city'|'service'`, `cities`, `services` | tabs / pills | loading, success | tab semantics | horizontal scroll mobile | homepage, search | indirect | `click_secondary_cta` | Selected mode changes target links |
| `OfficeCard` | Surface office answer | `office:Office`, `citySlug:string`, `serviceSlug?:string` | primary / nearby / generic | loading, partial, success, offline, error | address text not image; outbound links labelled | CTA stack on mobile | city-service, office directory, office page | indirect | `outbound_map_click` | Card shows office type, address, map CTA |
| `RequirementList` | Present prep list | `items:string[]`, `status:'verified'|'indicative'` | numbered / checklist | loading, partial, success | list semantics | no horizontal scroll | service page, city-service page | none | none | Docs list readable without accordion |
| `TrustBlock` | Show disclaimer, sources, verification date | `sourceRefs:SourceRef[]`, `verifiedAt:string`, `isIndicative:boolean` | compact / full | loading, partial, success, error | source links descriptive | full width all viewports | city page, service page, city-service, office, guide | none | `click_secondary_cta` | Always visible on critical pages |
| `SponsorBlock` | Paid local promotion | `sponsor:Sponsor`, `placement:string` | hero / inline / sidebar / sticky-mobile / newsletter | empty, success | disclosure label mandatory | sticky only on mobile when allowed | homepage, city-service, guide, office | direct | `sponsor_block_click` | Clearly labeled “Sponsorisé” |
| `AdSlot` | CLS-safe AdSense mount | `slotId:string`, `placement:string`, `minHeight:number` | hero-mid / inline / sidebar | loading, empty, success | `aria-label="Advertisement"` | hidden on small viewports when density rules require | homepage, city page, guide | direct | `ad_impression`, `ad_click_outbound` | Reserved space prevents layout shift |
| `CommercialCtaCard` | Non-ad hotel/transport/help offers | `title`, `body`, `href`, `category` | transport / translation / courier / planning | success | sponsored/commercial label when applicable | stack mobile | homepage, city-service | direct | `affiliate_cta_click` | Commercial intent clearly labeled |
| `FilterBar` | Narrow search and office directory | `cities`, `serviceTypes`, `officeTypes`, `selected` | sticky-desktop / inline-mobile | loading, success | controls grouped with labels | horizontal chip scroll mobile | search, office directory | indirect | `search_filter_apply` | Changes visible result count |
| `FormPanel` | Sponsor and issue intake | `kind:'issue'|'sponsor'`, `action:string` | issue / sponsor | loading, success, error | field labels, error summaries | single column | sponsor inquiry, report issue | direct for sponsor leads | `form_submit_start`, `form_submit_success`, `form_submit_error` | Required fields enforced server-side and client-side |
