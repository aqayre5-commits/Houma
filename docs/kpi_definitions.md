# kpi_definitions.md

## KPI Definitions
| KPI | Definition | Target Direction |
|---|---|---|
| task_start_rate | homepage primary CTA clicks / homepage views | up |
| city_service_completion_rate | outbound map clicks / city-service page views | up |
| search_success_rate | result-route clicks / search submits | up |
| sponsor_ctr | sponsor clicks / sponsor impressions | up |
| ad_ctr | ad outbound clicks / ad impressions | up |
| form_completion_rate | successful submissions / form starts | up |
| thin_page_ratio | thin or partial pages / total indexable pages | down |
| stale_page_ratio | stale critical pages / total critical pages | down |
| mobile_cta_visibility_pass_rate | QA passes / mobile CTA tests run | up |
