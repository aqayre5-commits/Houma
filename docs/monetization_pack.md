# monetization_pack.md

## AdSense Placements
| Surface | Placement | Rule |
|---|---|---|
| homepage | mid-content | after city cards |
| city pages | inline | after featured services |
| service pages | excluded in first viewport | sponsor only above fold |
| city-service pages | inline only | after summary and office card |
| office pages | sidebar desktop-only | hidden on mobile |
| guide pages | inline every 700–900 words equivalent | max 2 in v1 |

## Private Sponsor Placements
| Placement | Page Types | Rule |
|---|---|---|
| hero sponsor slot | homepage only | clearly labeled |
| in-content sponsor block | city-service, service | after core instruction module |
| sidebar sponsor card | office, guide desktop | not sticky |
| sticky mobile sponsor slot | city-service only | after CTA block |
| newsletter sponsor slot | guide only | below content |

## Commercial CTA Blocks
| Category | Use |

## Rules
1. Ad-safe spacing minimum 24px from core instruction block.
2. No deceptive layouts.
3. Clear sponsored labels.
4. Mobile-safe density.
5. CLS-safe rendering guidance with reserved slot heights.
